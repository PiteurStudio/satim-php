Generate payment link and check payment status using Satim.dz API

## Requirements

- PHP 8.0 or higher
- Satim.dz user account

## Installation

You can install the package via composer:

```bash
composer require piteurstudio/php-satim
```

## Usage

```php
$skeleton = new PiteurStudio\phpSatim();
```

## Testing

```bash
composer test
```
